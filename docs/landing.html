Why Notable

The tools you need, none of the friction.

Most notes apps are content management systems in a trench coat. Notable is just a great place to write — fast, organized, and quietly synced.

An editor that disappears

Headings, lists, checklists, code, tables, math. All the structure you want — and not a single floating toolbar in your peripheral vision.

Search that actually finds it

Full-text and fuzzy search across every note you've ever written. Typos forgiven. Tag filters, date filters, and instant results.

Sync without the spinner

Edit on your laptop, finish on your phone. Notable saves as you type and reconciles silently — no merge dialogs, no waiting.

Tags, not folders

Notes resist hierarchy. Tag them with as many ideas as fit, filter on intersections, and stop pretending your thoughts are tidy.

Your data, your keys

Notes are yours. Export to Markdown or JSON whenever. Self-host on your own server in an afternoon, or let us host it for you.

Built for keyboards

Every action has a shortcut. Slash-commands for blocks, ⌘K for jump-to-note, fold sections, drag without lifting your hands.

The editor

Words first. Everything else, when you ask.

Notable's editor is built on Tiptap — the same engine that powers Linear, GitLab, and a dozen tools you already use. Headings, lists, blockquotes, code blocks, tables, task lists, embeds. They're all there. None of them are in your way.

Slash commands. Type / anywhere to insert a heading, list, code block, callout, or table — no menus.
Markdown in, beautiful out. Type ## and it becomes a heading. Type - [ ] and you get a checkbox. Muscle memory preserved.
Code that looks like code. Syntax highlighting for 100+ languages, with a copy button you'll actually find.

Questions

Reasonable things to ask.

Can I self-host Notable?

Yes. Notable's server is a single Docker Compose file — Postgres, a Node API, and Caddy for TLS. Pull the image, set three environment variables, point a domain at it. Personal-use license is free; commercial self-hosting is included with the Team plan.

What happens to my data if I leave?

One click exports every note as Markdown, with all attachments, in a single zip. There's also a JSON export that preserves the full Tiptap document structure, if you want to migrate it somewhere structured. Your data is never held hostage.

Does Notable work offline?

Reading and editing existing notes works offline — the app keeps a local copy in your browser. Sync resumes silently when you're back online. Full offline-first sync with conflict resolution is coming in Notable 2.

How is search so fast?

Postgres full-text search with a GIN-indexed tsvector column, plus a pg_trgm index for typo tolerance. We're not running an external search service — it's just Postgres doing what Postgres does well. Most queries return in under 20ms even with tens of thousands of notes.

Is it really free, forever?

The free plan stays free. We make money on Personal and Team plans for people who want more storage, more devices, or shared workspaces. We will not run ads, sell data, or invent a paywall around features you already use.
